# VBA Custom Website (Next.js)

## Stack
- Next.js (App Router)
- Tailwind CSS
- Supabase (ready)
- Vercel deployment

## Getting Started (Local)
1. npm install
2. npm run dev

## Environment Variables (.env.local)
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key

## Deploy to Vercel
- Import Git repo
- Add env vars
- Deploy
